# 相機與裁剪參數標準作業程序

Casino Camera SOP - 賭場相機設置與撲克辨識系統標準作業程序文檔

## 專案簡介

這是一個統一的標準作業程序（SOP）文檔系統，包含了賭場相機設置、撲克辨識系統配置和影像裁剪參數的完整操作指南。所有文檔提供中英文雙語支援。

## 文檔結構

```
casino-camera-sop/
├── index.html                          # 主導航頁面
├── assets/
│   ├── css/
│   │   └── styles.css                  # 統一樣式表
│   ├── js/
│   │   └── language.js                 # 語言切換功能
│   └── images/                         # 圖片資源
└── pages/
    ├── recognition-camera.html         # 辨識相機設定 SOP
    ├── recognition-camera-xnp.html     # XNP辨識相機設定 SOP
    ├── angle-adjustment-xnp.html       # XNP相機角度調整 SOP
    ├── poker-configuration.html        # 撲克配置 SOP
    ├── poker-calibration.html          # 撲克辨識校準 SOP
    └── crop-parameters.html            # 裁剪參數 SOP
```

## 文檔內容

### 📷 辨識相機設定 SOP
靜態相機設定的完整指南，包括視頻、網絡、圖像和曝光設定。

### 📷 XNP辨識相機設定 SOP
Hanwha XNP-6320 PTZ相機的靜態設定標準作業程序。

### 🔧 XNP相機角度調整 SOP
Hanwha XNP-6320 PTZ相機的角度調整詳細步驟。

### 🎴 撲克配置 SOP
撲克辨識系統的配置設定指南。

### 🎯 撲克辨識校準 SOP
撲克辨識系統的校準程序和驗證步驟。

### ✂️ 裁剪參數 SOP
影像裁剪和標記的完整流程，用於辨識系統訓練。

## 功能特色

- 🌐 **雙語支援**：中文/英文即時切換
- 📱 **響應式設計**：支援桌面和移動設備
- 🎨 **統一風格**：一致的視覺設計和排版
- 🔍 **易於導航**：清晰的結構和導航系統
- 📝 **詳細說明**：完整的配置參數和操作步驟

## 使用方式

1. 開啟 `index.html` 作為入口
2. 使用右上角的語言切換按鈕選擇中文或英文
3. 點擊卡片進入對應的 SOP 文檔
4. 按照步驟進行配置或操作
